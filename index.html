How to Use

  1. Adjust the number of rows and columns.
  2. Set the gap between grid items.
  3. Click "Generate Grid" to see your layout.
  4. You can draw any type of grid layout.
  5. Copy the generated HTML, CSS, or React code.

Tip: You can also clear the grid and start over at any time.
